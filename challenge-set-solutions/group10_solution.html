Question 1

Solve for nnn in the following:

  • n div 7=5 , n mod 7=1n \text{ div } 7 = 5 \text{ , } n \text{ mod } 7 = 1n div 7=5 , n mod 7=1
  • n div 6=−6 , n mod 6=2n \text{ div } 6 = -6 \text{ , } n \text{ mod } 6 = 2n div 6=−6 , n mod 6=2
  • n div 5=7 , n mod 5=3n \text{ div } 5 = 7 \text{ , } n \text{ mod } 5 = 3n div 5=7 , n mod 5=3

Answer:

  • 7∗5+1=367*5 + 1 = 367∗5+1=36
  • −6∗6+2=−34-6*6+2 = -34−6∗6+2=−34
  • 5∗7+3=385*7 +3 = 385∗7+3=38

Note, for n div m=p , n mod m=qn \text{ div } m = p \text{ , } n \text{ mod } m = qn div m=p , n mod m=q that n=m⋅p+qn = m \cdot p + qn=m⋅p+q where ppp is the quotient (integer division) and qqq is the remainder.

Question 6

Use the following process with the table below:

  1. Cross out the box with 1.
  2. Goto the box with 2.
  3. Let nnn equal the box you are on. Cross out all boxes that are multiples of nnn starting with n2n^2n2.
  4. Goto the smallest box greater than nnn that is not crossed out and go back to step three. If there are no more available numbers then stop.

Consider the numbers that are not crossed out. What do they represent?

Answer: The prime numbers less than or equal to 100. This is called the Sieve of Eratosthenes.
